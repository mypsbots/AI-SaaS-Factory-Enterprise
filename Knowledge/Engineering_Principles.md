# KNOWLEDGE — Engineering Principles

> The cross-cutting principles every ASFE artefact must uphold.

---

## 1. Summary

The cross-cutting principles every ASFE artefact must uphold.

---

## 2. Context

Curated reference knowledge for teams building SaaS products with ASFE.

---

## 3. Key Concepts

- Security and Privacy by Design
- Accessibility by Default
- Cloud Native and API First
- Observability First
- Zero Trust and Least Privilege
- Documentation and Infrastructure as Code

---

## 4. Guidance

- Apply principles as defaults, not exceptions.
- Make principles measurable via gates and KPIs.
- Resolve conflicts in favour of security, privacy, and safety.

---

## 5. Pitfalls

- Treating principles as aspirational slogans.
- Bolting on security/accessibility after build.
- Principles without enforcement.
