# KNOWLEDGE — Security Reference Architecture

> Baseline secure architecture patterns for SaaS.

---

## 1. Summary

Baseline secure architecture patterns for SaaS.

---

## 2. Context

Curated reference knowledge for teams building SaaS products with ASFE.

---

## 3. Key Concepts

- Zero-trust network and identity
- Defense in depth
- Secrets and key management
- Tenant isolation
- Secure SDLC and supply chain
- Encryption everywhere

---

## 4. Guidance

- Default deny; segment and isolate by tenant.
- Centralise identity, secrets, and key management.
- Automate security gates across the SDLC.

---

## 5. Pitfalls

- Flat networks and shared credentials.
- Implicit trust between services.
- Manual, late security reviews.
